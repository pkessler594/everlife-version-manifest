# Everlife 2 Version Manifest

This public repo hosts the small JSON file read by Everlife 2 on launch.

Update `version-manifest.json` after a new App Store / Google Play build is live.

Current live manifest:

```json
{
  "latestIosBuild": "0.9.17",
  "latestAndroidBuild": "107"
}
```

The game intentionally uses iOS build number and Android bundle version code only.
